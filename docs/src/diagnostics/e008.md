# E008: Inference Timeout

**Severity:** Warning

```
type inference timed out -- missing types for: `bigFunction`, `helper`
```

Type inference for this file exceeded the configured deadline. Bindings that were inferred before the deadline still have their types; only the remaining bindings are affected.

## Common causes

- Very large files with many bindings or deeply nested expressions.
- Highly polymorphic recursive functions that cause the inference engine to explore many constraint paths.
- The default deadline (10 seconds) is too short for the file's complexity.

## How to fix

- Increase the deadline in `tix.toml`:
  ```toml
  deadline = 30   # seconds (default: 10)
  ```
- Split large files into smaller modules. This also improves incremental re-checking.
- Add type annotations to complex bindings to reduce the work the inference engine needs to do.
